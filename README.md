# Bet-2000 Live — Quick Start

Einfacher Start (automatisch Tunnel, falls nötig):

1. Stelle sicher, dass Node.js installiert ist.
2. (Optional) Setze `NGROK_AUTHTOKEN` in der Umgebung, falls du ngrok bevorzugst.
3. Starte alles mit:

```bash
./easy-start.sh
```

Die Seite `Live-Control.html` dient als Steuerzentrale (Code: `2409`). Die Spielseite ist `Bet-2000-Schwalbkraiburg.html`.

Discovery: `http://localhost:8080/public-ws` liefert die öffentliche `ws` URL.
